# miniRedis Documentation

miniRedis is a simplified, asynchronous, in-memory Redis-like server built with Rust and Tokio. It implements a subset of the Redis RESP protocol, supports concurrent TCP clients, and provides TTL expiration, approximate LRU eviction, and runtime configuration.
